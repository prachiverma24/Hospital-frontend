This is a capstone-level starter scaffold for a Healthcare Appointment & Records Portal built with React.

Features included:
- React Router v7 routing
- Authentication with Context API
- React Query setup for fetching mock data
- Lazy-loaded pages and components
- Accessible multi-step booking form (starter)
- Reusable UI components and custom hooks

How to run
1. Install dependencies: `npm install` or `yarn`
2. Start dev server: `npm run start` (uses Vite)

Folder layout (src)
- src/
  - components/  (Button, Input, Card, Loader)
  - context/     (AuthContext)
  - hooks/       (useAuth, data hooks)
  - pages/       (login, register, dashboard, search, book, appointments, profile, records)
  - services/    (mock API services)
  - App.jsx, main.jsx

This is a starter scaffold. Expand mock services to real APIs and add styles as needed.
